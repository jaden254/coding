Complete HTML Study Notes: From Basics to Semantics

A comprehensive guide to HyperText Markup Language (HTML).

1. Introduction to HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. It describes the structure of a web page using markup.

Basic Structure:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document Title</title>
</head>
<body>
    <h1>Visible Content Here</h1>
</body>
</html>

2. Common HTML Elements

  • <h1> to <h6>: Headings (H1 is the most important)
  • <p>: Paragraphs
  • <a>: Links (uses the href attribute)
  • <img>: Images (uses src and alt attributes)
  • <ul>, <ol>, <li>: Unordered lists, ordered lists, and list items
  • <div>: A division or section (used for layout, non-semantic)
  • <span>: An inline container for text

3. Attributes

Attributes provide additional information about elements. They are always included within the start tag and usually come in name/value pairs (e.g., name="value").

  • href: Specifies the URL for a link (<a href="url">).
  • src: Specifies the path to an image (<img src="path">).
  • alt: Provides alternative text for an image (accessibility).
  • class: Used to define CSS styles for elements.
  • id: Specifies a unique ID for an element (for CSS or JavaScript).

4. Semantic HTML5

Semantic elements describe their meaning to the browser and the developer, leading to better accessibility and SEO.

  • <header>: Defines a header for a document or section.
  • <nav>: Defines a set of navigation links.
  • <main>: Specifies the main content of a document.
  • <article>: Defines independent, self-contained content.
  • <section>: Defines a section in a document.
  • <aside>: Defines content aside from the content (like a sidebar).
  • <footer>: Defines a footer for a document or section.

5. Forms

Forms are used to collect user input.

<form action="/submit_page" method="POST">
    <label for="fname">First name:</label><br>
    <input type="text" id="fname" name="fname"><br>
    <input type="submit" value="Submit">
</form>

Key input types include: text, password, submit, checkbox, radio, date.
