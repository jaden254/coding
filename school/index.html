üìò HTML Notes

HTML Boilerplate

Basic structure of an HTML document.

<!DOCTYPE html>
        <html lang="en">
          <head>
            <meta charset="UTF-8">
            <title>Page Title</title>
          </head>
          <body>
            <h1>Hello World</h1>
          </body>
        </html>
Head Tags

Metadata and links go inside the <head> tag.

<head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1">
          <link rel="stylesheet" href="style.css">
        </head>
Semantic HTML

Use meaningful tags for better accessibility and SEO.

<header>...</header>
        <nav>...</nav>
        <main>...</main>
        <footer>...</footer>
HTML Lists

Use ordered or unordered lists for structured content.

<ul>
          <li>Item 1</li>
          <li>Item 2</li>
        </ul>
        
        <ol>
          <li>First</li>
          <li>Second</li>
        </ol>
Links and Images

Use anchor and image tags to add navigation and visuals.

<a href="https://example.com">Visit Site</a>
        <img src="image.jpg" alt="Description">
HTML Forms

Collect user input with form elements.

<form>
          <input type="text" placeholder="Name">
          <input type="email" placeholder="Email">
          <button type="submit">Send</button>
        </form>
HTML Tables

Display tabular data using rows and columns.

<table>
          <tr><th>Name</th><th>Age</th></tr>
          <tr><td>Jaden</td><td>20</td></tr>
        </table>
Div and Span

Use <div> for block-level and <span> for inline grouping.

<div>This is a block</div>
        <span>This is inline</span>

üé® CSS Notes

Basic Styling

Apply color, font, and alignment to elements.

h1 {
          color: #333;
          font-family: Arial, sans-serif;
          text-align: center;
        }
CSS Selectors

Target elements using tags, classes, or IDs.

/* Tag selector */
        p { color: blue; }
        
        /* Class selector */
        .card { padding: 20px; }
        
        /* ID selector */
        #header { background-color: black; }
Box Model

Understand padding, border, and margin around elements.

div {
          padding: 10px;
          border: 2px solid #ccc;
          margin: 20px;
        }
Flexbox Layout

Align and distribute items in a flexible container.

.container {
          display: flex;
          justify-content: space-between;
          align-items: center;
        }
CSS Grid

Create complex layouts using rows and columns.

.grid {
          display: grid;
          grid-template-columns: 1fr 2fr;
          gap: 20px;
        }
Responsive Design

Adapt styles for different screen sizes.

@media (max-width: 768px) {
          h1 {
            font-size: 1.5rem;
          }
        }
CSS Variables

Define reusable values for colors, fonts, etc.

:root {
          --main-color: #ff3c38;
        }
        
        button {
          background-color: var(--main-color);
        }
Transitions & Hover

Add smooth animations when hovering over elements.

a {
          color: #000;
          transition: color 0.3s ease;
        }
        
        a:hover {
          color: #ff3c38;
        }
Z-Index & Position

Control stacking and placement of elements.

.box {
          position: absolute;
          top: 50px;
          left: 100px;
          z-index: 10;
        }

DOM Manipulation

Change HTML content dynamically.

Event Listener

React to user actions like clicks.

document.querySelector("button").addEventListener("click", () => {
          alert("Button clicked!");
        });
Conditionals

Make decisions based on logic.

if (age >= 18) {
          console.log("Adult");
        } else {
          console.log("Minor");
        }
Loops

Repeat actions using for or while.

for (let i = 0; i < 5; i++) {
          console.log("Count:", i);
        }
Fetch API

Get data from external sources.

fetch("https://api.example.com/data")
          .then(res => res.json())
          .then(data => console.log(data));
