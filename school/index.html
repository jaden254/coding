üìù Notes for Coding

HTML Boilerplate

A basic HTML document structure to start with.

<!DOCTYPE html>
    <html lang="en">
    <head>
        <meta charset="UTF-8">
        <title>Page Title</title>
    </head>
    <body>
        <h1>Hello World</h1>
    </body>
    </html>
CSS for a Heading

Example of how to center and style a heading with CSS.

h1 {
        color: #333;
        font-family: Arial, sans-serif;
        text-align: center;
    }
JavaScript Function

A simple JavaScript function to display an alert message.

function greet() {
        alert("Welcome to the website!");
    }
Bootstrap Grid Example

An example of how Bootstrap columns work for a responsive layout.

<div class="row">
      <div class="col-md-6">Column 1</div>
      <div class="col-md-6">Column 2</div>
    </div>
Semantic HTML

Using meaningful tags like `header`, `nav`, and `main` to improve accessibility and SEO.

<header>
      <nav>...</nav>
    </header>
    <main>
      <section>...</section>
    </main>
    <footer>...</footer>
CSS Flexbox

A modern way to lay out items in a container, allowing for flexible and responsive designs.

.flex-container {
      display: flex;
      justify-content: center;
      align-items: center;
    }
ES6 Arrow Function

A concise syntax for writing JavaScript functions, useful for simple callbacks.

const add = (a, b) => a + b;
    console.log(add(2, 3)); // 5
CSS Variables

Define custom properties in CSS to easily reuse values throughout your stylesheets.

:root {
      --primary-color: #ff3c38;
    }
    .btn-primary {
      background-color: var(--primary-color);
    }
Event Listener

Use JavaScript to listen for user actions, such as a button click.

document.getElementById("myBtn").addEventListener("click", () => {
        alert("Button clicked!");
    });
Accessible Images

Always include the `alt` attribute on images for screen readers and search engines.

<img src="image.jpg" alt="A descriptive text for the image">
CSS Grid

A powerful layout system for creating complex, two-dimensional page structures.

.grid-container {
        display: grid;
        grid-template-columns: 1fr 1fr 1fr;
        gap: 10px;
    }
Ternary Operator

A concise way to write a conditional statement in a single line.

const result = (score > 60) ? "Pass" : "Fail";
HTML Lists

Organize content with ordered (`<ol>`) or unordered (`<ul>`) lists for better structure.

<ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
CSS Pseudo-classes

Apply styles based on a special state of an element, like `:hover` or `:focus`.

a:hover {
        text-decoration: underline;
    }
