Learn using Notes

⚙️ JavaScript Notes

Alert Message

Display a simple popup message.

alert("Hello, world!");
Declaring Variables

Use let, const, or var to store values.

let name = "Jaden";
const age = 20;
var city = "Nairobi";
Function Declaration

Reusable blocks of code for specific tasks.

function greet() {
  console.log("Welcome!");
}
Arrow Function

Shorter syntax for writing functions.

const add = (a, b) => a + b;
console.log(add(2, 3)); // 5
DOM Manipulation

Change HTML content dynamically.

document.getElementById("demo").innerText = "Updated text!";
Event Listener

React to user actions like clicks.

document.querySelector("button").addEventListener("click", () => {
  alert("Button clicked!");
});
Conditionals

Make decisions based on logic.

if (age >= 18) {
  console.log("Adult");
} else {
  console.log("Minor");
}
Loops

Repeat actions using for or while.

for (let i = 0; i < 5; i++) {
  console.log("Count:", i);
}
Fetch API

Get data from external sources.

fetch("https://api.example.com/data")
  .then(res => res.json())
  .then(data => console.log(data));
